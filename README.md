# Online-Misinformation-Detection-System
A comparative study of rule-based and machine learning approaches for online misinformation detection using NLP and TF-IDF feature extraction.
# Project Overview
This project investigates the use of Natural Language Processing (NLP), rule-based detection, and machine learning techniques for identifying misinformation in online text. The system analyses textual content and classifies it as either truthful or misleading.
# Objectives
Develop a rule-based misinformation detection module.
Implement a machine learning classifier using TF-IDF features.
Compare the performance of both approaches.
Evaluate the feasibility of lightweight misinformation detection systems.
